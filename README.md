# Guess the Number Game

## Project Description
This project is a simple interactive web game where the player tries to guess a randomly generated number. The game gives feedback after each guess, such as whether the guess is too high or too low. The player wins by guessing the correct number before running out of attempts.

## Basic Features
- Random number generation
- Input box for guesses
- Submit button
- Feedback message (too high / too low / correct)

## Enhanced Features
- Score tracking
- Limited number of attempts
- Hint system

## Advanced Features
- Difficulty levels
- Persistent leaderboard using localStorage or backend
- Restart game option
